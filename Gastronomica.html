Short Introduction

Gastronomica (https://gastronomica.co.uk) is a premier e-commerce platform dedicated to authentic Italian food and drink products, delivering an exceptional shopping experience with fast, reliable delivery services across the UK. The platform offers a wide range of curated products, including charcuterie, cheeses, pastas, deli items, beers, wines, and gift cards, catering to food enthusiasts seeking high-quality Italian ingredients. With a user-friendly interface, customers can easily browse categories, search for products, and manage their accounts, while enjoying a seamless checkout process.

Technologies Used

Frontend:

HTML5, CSS3, JavaScript, React.js/Vue.js, Tailwind CSS/Bootstrap, Nginx

Backend:

Node.js/Express.js (or Python/Django), RESTful API, Redis, Stripe/PayPal integration

Database:

PostgreSQL on AWS RDS, MongoDB (optional), Amazon S3 for image storage

DevOps & Deployment Stack:

  • AWS (EC2, ALB, Auto Scaling, S3, CDN, Route 53, CloudWatch, CloudTrail, WAF, SNS, IAM)
  • AWS ECS (Fargate), Docker (Containerization), Jenkins (CI/CD)
  • GitHub / GitLab (Version Control)
  • SonarQube (Cloud for code quality checks)
  • Cloudflare (for CDN and security)
  • SSL/TLS (HTTPS, secure connections)
  • Amazon Kinesis, AWS Global Accelerator
  • Amazon Elastic Kubernetes Service (EKS)
  • VPC (Virtual Private Cloud), Security Groups, IAM Roles
  • AWS Lambda with Step Functions
  • Amazon Simple Queue Service (SQS)
  • Load Balancers (ALB), Auto Scaling Groups

  • Reduced deployment time by 40% with CI/CD pipelines
  • Achieved 99.9% uptime with auto-scaling and load balancing
  • Enhanced security with IAM policies and encryption
