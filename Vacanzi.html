Vacanzi

South India's Leading AI-Powered Job Platform – Land Your Dream Job in Just 30 Minutes!

Short Introduction

Vacanzi is an advanced job portal — faster and smarter than traditional platforms like Naukri and LinkedIn. Connect with professional counselors, network with friends, enjoy audio/video calls, form groups, take AI-driven mock interviews, and even watch fun reels! With a modern UI/UX and presence on both web and mobile, Vacanzi gained over 1 lakh users in just one month!

Feature List

  • Secure User Login (Mobile authentication)
  • AI-Powered Interviews
  • Fun Reels for Relaxation
  • Latest Jobs Shown First
  • AI-Based Job Recommendations
  • Immediate Company Response
  • Land Your Dream Job in Just 30 Minutes!
  • Modern, Beautiful UI/UX
  • Web and Mobile App Availability
  • Audio/Video Calling, Group Creation

Technologies Used

Frontend:

React.js, HTML5, CSS3, JavaScript (ES6+), Nginx, Responsive Design (Bootstrap, Tailwind CSS)

Backend:

Node.js, Express.js, FFmpeg, Redis, RESTful API

Database:

MongoDB, MongoDB Atlas, AWS RDS

DevOps & Deployment Stack:

  • AWS (EC2, ALB, Auto Scaling, S3, CDN, Route 53, CloudWatch, CloudTrail, WAF, SNS, IAM)
  • AWS ECS (Fargate), Docker (Containerization), Jenkins (CI/CD)
  • GitHub / GitLab (Version Control)
  • SonarQube (Cloud for code quality checks)
  • Cloudflare (for CDN and security)
  • SSL/TLS (HTTPS, secure connections)
  • VPC (Virtual Private Cloud), Security Groups, IAM Roles
  • Load Balancers (ALB), Auto Scaling Groups
